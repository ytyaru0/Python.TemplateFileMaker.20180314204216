# 守備範囲

* 単一テンプレファイル
* 複数ファイル、ディレクトリ

## 範囲外

* プロジェクト一式は`proj`でやる
* リポジトリ作成は`repo`でやる

以下のようなコマンドを想定している。このうち`y`のもののみ対象。

コマンド|処理概要
--------|--------
`$ y {lang} {context}`|単一ファイル
`$ pj {lang} {context}`|相対パス関係がある複数ファイルとディレクトリ。
`$ repo -l {license} {lang} {context}`|`pj` + README, LICENSE, .gitignore
